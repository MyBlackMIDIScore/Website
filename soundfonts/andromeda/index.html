Andromeda Steinway Grand – Remastered

This is a very old soundfont project by MBMS and MusiMasta based on the Steinway Grand Piano library from Apple’s GarageBand DAW. So after a request from someone in the BMC, I decided to remake this soundfont. This time although I did not use GarageBand, because I don’t own an Apple computer (last time the samples were provided by MusiMasta). So I used a free piano library called Keyzone which included this sound and then edited the samples one by one to achieve this high quality sound. This soundfont features an early version of the Zephyr Enhancement System. It is in standard SF2 format using 20 dynamic velocity layers optimized for Black MIDI playback.

You can download a copy of this soundfont using the button located below this text. Please read the Terms of Use for this soundfont, which can be found right under the download button.
