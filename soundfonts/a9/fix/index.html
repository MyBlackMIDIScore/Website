How to apply the A9 Fixed Presets

One of the known issues of the A9 Remastered soundfont, is that the “Random Offset” presets did not produce any sound. This has been present since the initial release of the soundfont. I am not planning to create updates for this specific soundfont (as I am doing with Amethyst Imperial Grand) so I am going to replace the main download link with a new one that contains the full soundfont with proper working presets.

I understand that some people do not have the time to re-download the entire soundfont just to fix some presets so I will also release the following ZIP file (compressed folder) which contains only the presets that need to be replaced/fixed. The tutorial found below will guide you through this procedure so you can get the Random Offset presets working in no time!

WARNING: This is only for people who already have a copy of the A9 Remastered soundfont. If you have not downloaded the soundfont before, please visit the official download page here: https://mbms.me/soundfonts/a9

Tutorial

Step 1

Download the A9 Fixed Presets folder using the following link: https://download.mbms.me/A9FixedPresets.zip

Step 2

Navigate to the folder in which you store the A9 Remastered soundfont and delete the folder called “Presets”.


Step 3

Open the ZIP file you just downloaded and copy the “Presets” folder from the ZIP to the A9 folder.


That’s it

You can now use the Random Offset presets normally in OmniMIDI or any other soundfont player that supports A9!
