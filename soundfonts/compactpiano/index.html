Compact Piano Lite

This soundfont was my attempt to create a very small in size sound library, the sound of which does not sound like most other small soundfonts. The total size of it is 8MB! I would not call this a professional soundfont by any means, but it was a nice experiment and is definitely optimal for people with limited disk space that want some good and clear sound. It is fairly old and was created with a mixture of different Yamaha piano libraries with some extra bass feel, in order to be used for Black MIDIs. The preset format is Cakewalk SFZ and the samples are in OGG format with 48.000kHz and about 1500kbps bit rate. It features the standard 20 dynamic velocity layers for extra realism and clearance.

You can download a copy of this soundfont using the button located below this text. The file is distributed via my archive domain (download.mbms.ml). Please read the Terms of Use for this soundfont, which can be found right under the download button.
